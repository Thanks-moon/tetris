# 能量爆裂俄罗斯方块

一个单文件浏览器版俄罗斯方块小游戏。核心玩法采用经典 10 x 20 棋盘，并在消行时加入能量爆裂效果：白热闪光、横向能量波、粒子飞散和轻微屏幕震动。

## 在线/本地运行

直接用浏览器打开项目中的 HTML 文件即可：

```text
tetris-energy-clear.html
```

不需要安装依赖，也不需要启动开发服务器。

## 操作方式

| 按键 | 功能 |
| --- | --- |
| ← / → | 左右移动 |
| ↓ | 软降 |
| ↑ | 旋转 |
| Space | 硬降 |
| P | 暂停/继续 |
| R | 重新开始 |

## 功能特性

- 标准俄罗斯方块棋盘和七种方块
- 随机袋算法生成方块
- 分数、等级、消行数和连击统计
- 下一块预览
- 幽灵落点提示
- 消行能量爆裂动画
- 单 HTML 文件交付，方便分享和部署
